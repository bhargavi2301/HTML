    11. What is the difference between a block-level element and an inline element ?

    block-level element:A block-level element always starts on a new line, and the browsers automatically add some space (a margin) before and after the element. A block-level element always takes up the full width available (stretches out to the left and right as far as it can). Two commonly used block elements are: < p > and < div > .

    inline element:Inline elements display in a line.

    17. What is SVG?

    SVG stands for Scalable Vector Graphics. It is a widely-used XML-based vector image format for two-dimensional graphics with support for interactivity and animation. SVG files are used to define vector-based graphics for the web and can be scaled up or down in size without losing image quality, making them ideal for responsive web design.

    18.What is difference between HTML and XHTML?

    HTML (Hypertext Markup Language) and XHTML (Extensible Hypertext Markup Language) are both markup languages used for creating web pages, but they have some key differences:
    HTML: HTML has more lenient syntax rules. It allows for certain mistakes like unclosed tags or mixed case tag names, and the document will still render in most browsers.
    XHTML: XHTML is more strict. It follows the rules of XML, which means that all tags must be properly nested, closed, and in lowercase. Even a small error can cause the entire page to break.
